Home/Dashboard functions as the central hub, providing access to real-time updates, task notifications, and recent activities for streamlined platform navigation. Users receive notifications upon task completion, system updates, and relevant news. Moreover, users can access use cases and a usage summary that outline how CAMEO services have been applied in real-life situations, offering a comprehensive overview of practical scenarios. Map alerts will be added in the future, enhancing the platform's capabilities.
Map Explorer provides an engaging interactive map interface, empowering users to delve into geographical data layers and satellite imagery. Alongside this, users benefit from a robust search tool allowing queries based on date range, location, and satellite platform. Additionally, users can apply common filters such as satellite platform, processing level, resolution, data type, and cloud cover percentage. To enhance the user experience, Map Explorer incorporates standard map functionalities like zoom, scale adjustments, latitude and longitude tracking, polygon/AOI (Area of Interest) creation, layer data value inspection, legend display, and the option to view or hide layers. Furthermore, the platform offers temporal filtering options, enabling users to select data from specific time periods or ranges, thereby providing a versatile and comprehensive exploration experience.

DAG Maker constructs the graph (the DAG), gathers the various containerised tasks required, and then runs them in an orchestration system such as Apache Airflow.
